Quick Note Taker

Extensão para Chrome (Manifest V3) — Projeto do Bootcamp.

Uma forma simples e rápida de guardar anotações diretamente no seu navegador.

Download da Extensão

O que ela faz?

A Quick Note Taker permite que você faça anotações em um popup prático. Tudo que você digita é salvo automaticamente. Você também pode selecionar qualquer texto em uma página, clicar com o botão direito e adicionar diretamente às suas notas!

Instalação

Como esta extensão não está na Chrome Web Store, a instalação é feita manualmente no modo de desenvolvedor.

  1. Baixe a versão mais recente: Faça o download do arquivo .zip da página de Releases.
  2. Descompacte o arquivo: Extraia o conteúdo do .zip em uma pasta permanente no seu computador.
  3. Abra as Extensões do Chrome: Navegue para chrome://extensions.
  4. Ative o Modo de Desenvolvedor: Ligue o interruptor no canto superior direito.
  5. Carregue a Extensão: Clique em "Carregar sem compactação" e selecione a pasta que você acabou de descompactar.

Pronto! A extensão estará ativa.

Permissões Utilizadas

  • storage: Para salvar suas anotações de forma segura no navegador.
  • contextMenus: Para criar a opção "Adicionar à Anotação Rápida" no menu de clique direito.
  • alarms: Usado para demonstrar eventos agendados no service worker.
